# 💰💰💰 题

给你一个整数数组 nums，返回 数组 answer ，其中 answer[i] 等于 nums 中除 nums[i] 之外其余各元素的乘积 。

题目数据 保证 数组 nums 之中任意元素的全部前缀元素和后缀的乘积都在 32 位 整数范围内。

请 不要使用除法，且在 O(n) 时间复杂度内完成此题。

### 示例 1:

输入: nums = [1,2,3,4]
输出: [24,12,8,6]

### 示例 2:

输入: nums = [-1,1,0,-3,3]
输出: [0,0,9,0,0]

# 💯💯💯

#### 整体思路

- **为什么不用除法？** 题目禁止使用除法，同时需要避免潜在的零除问题（如果数组有零）。
- **核心方法：前缀乘积 + 后缀乘积**
  - 第一遍遍历（从左到右）：计算每个位置的前缀乘积（即当前位置左侧所有元素的乘积），并存入 `answer` 数组。
  - 第二遍遍历（从右到左）：使用一个变量 `right` 维护后缀乘积（当前位置右侧所有元素的乘积），边遍历边乘到 `answer` 中。
- **时间复杂度**：两次 O(n) 遍历，总 O(n)。
- **空间复杂度**：O(1) 额外空间（不计输出数组）。
- **注意事项**：处理包含零或负数的数组时，乘积会自然为零或带负号；数组长度至少为 2（题目保证）。

#### 代码逐行分析

```javascript
/**
 * @param {number[]} nums
 * @return {number[]}
 */
var productExceptSelf = function (nums) {
  const n = nums.length; // 获取数组长度
  const answer = new Array(n).fill(1); // 初始化 answer 为全 1 数组，作为前缀乘积的起点

  // 第一遍：计算前缀乘积
  for (let i = 1; i < n; i++) {
    answer[i] = answer[i - 1] * nums[i - 1]; // answer[i] = nums[0] * ... * nums[i-1]
  }

  // 第二遍：计算后缀乘积，使用一个变量从右到左累乘
  let right = 1; // 初始化右后缀为 1（最右边无元素）
  for (let i = n - 1; i >= 0; i--) {
    answer[i] *= right; // 乘上右侧乘积：answer[i] = 前缀 * 后缀
    right *= nums[i]; // 更新 right 为下一个位置的后缀（right = right * nums[i]）
  }

  return answer; // 返回结果数组
};
```

#### 结合案例学习：nums = [1, 2, 3, 4]

让我们用这个简单数组逐步模拟执行过程，帮助理解代码的运行逻辑。预期输出为 [24, 12, 8, 6]（因为 2*3*4=24, 1*3*4=12, 1*2*4=8, 1*2*3=6）。

1. **初始化**：

   - n = 4
   - answer = [1, 1, 1, 1]

2. **第一遍遍历（前缀乘积，从左到右）**：

   - i=1: answer[1] = answer[0] _ nums[0] = 1 _ 1 = 1
   - i=2: answer[2] = answer[1] _ nums[1] = 1 _ 2 = 2
   - i=3: answer[3] = answer[2] _ nums[2] = 2 _ 3 = 6
   - 此时 answer = [1, 1, 2, 6]（分别代表左侧乘积：空=1, 1, 1*2=2, 1*2\*3=6）

3. **第二遍遍历（后缀乘积，从右到左）**：
   - right = 1（起始）
   - i=3: answer[3] _= right → 6 _ 1 = 6；然后 right _= nums[3] → 1 _ 4 = 4
   - i=2: answer[2] _= right → 2 _ 4 = 8；然后 right _= nums[2] → 4 _ 3 = 12
   - i=1: answer[1] _= right → 1 _ 12 = 12；然后 right _= nums[1] → 12 _ 2 = 24
   - i=0: answer[0] _= right → 1 _ 24 = 24；然后 right _= nums[0] → 24 _ 1 = 24（但循环结束）
   - 最终 answer = [24, 12, 8, 6]

通过这个案例，可以看到前缀提供左侧信息，后缀提供右侧信息，二者相乘即得结果。如果数组有零，例如 nums = [1, 0, 3, 4]，则输出 [0, 36, 0, 0]（因为包含零的位置乘积为零，其他位置需排除零）——代码会自动处理。

#### 另一个案例：包含零和负数 nums = [-1, 1, 0, -3, 3]

- 第一遍前缀：answer = [1, -1, -1, 0, 0]
- 第二遍后缀：
  - i=4: 0 \* 1 = 0, right=3
  - i=3: 0 _ 3 = 0, right=3_(-3)=-9
  - i=2: -1 * (-9) = 9, right=-9*0=0
  - i=1: -1 * 0 = 0, right=0*1=0
  - i=0: 1 _ 0 = 0, right=0_(-1)=0
- 输出：[0, 0, 9, 0, 0]（验证正确：排除自身后的乘积）。

这个算法高效且鲁棒，适合实际应用。如果需要测试更多案例，可以在 JS 环境中运行函数验证。
